利用hugo搭建博客，并挂载到github

时间： 2022-03-13   |   分类： 软件安装   | 字数： 1736 字 | 阅读： 4分钟 | 阅读次数：

突然想搭建一个自己的个人博客，记录下自己的点滴收获与成长，时时擦拭。

下面是我的配置过程

安装git

1.官网下载对应版本安装包，Windows的以.exe结尾

2.安装

参考：Git安装教程（Windows安装超详细教程）

3.配置

在命令行输入

1
2
3
4
5
6
7
8

# 创建本地密匙
ssh-keygen -t rsa -C "email@xxx.com" # 双引号内容为你自己注册账号时的邮箱
cat /c/Users/22079/.ssh/id_rsa.pub # 得到的密匙复制到github设置中的 SSH and GPG keys
# 验证是否链接成功
ssh -T git@github.com
# 全局配置
git config --global user.name "Your Name"
git config --global user.email "email@example.com"

--global参数表示会在这台机器上的所有git仓库使用该配置

安装hugo

本文在windows环境下写成

1）到github仓库下载对应版本

2）解压，得到.exe文件

3）将.exe文件配置到PATH

Windows 搜索环境变量，将.exe文件路径保存到环境变量PATH

使用hugo生成博客

1）新建博客

进入想要放置博客的目录，右键点击git bash here， 输入hugo new site blog，可得到如下文件目录：

1
2
3
4
5
6
7
8

blog
├── archetypes	# 新文章默认模板
├── config.toml	# hugo配置文档
├── content	# 存放markdown格式文章
├── data	# 生成网站可用的数据文件，可用在模版中
├── layouts	# 生成网站时可用的模板
├── static	# 存放图像、CNAME、css、js等，发布后该目录下所有资源将处于网页根目录
└── themes	# 存放下载的主题

使用下面的命令生成新的文章草稿：

1

hugo new posts/first-post.md

2）下载主题

在官网Hugo Themes选择想要的主题，会链接到github，下载相应主题的zip文件，解压后放到blog的themes文件夹下

3）生成网站

根据不同主题的使用说明，进行操作，eg:将examplesite文件里的内容替换到blog里

现在使用如下命令建立本地服务器：

1

hugo server

并在浏览器中输入网址http://localhost:1313/，查看网页效果

如果觉得没有问题了便可以使用如下命令：

1

hugo

如此一来网页便生成在默认的public子目录中了。

当然更建议另一种方式，将网页生成在doc目录中

1
2
3
4
5

# 将下面一行插入配置文件config.toml中
publishdir = "doc"	# toml
publishDir = docs	# yaml
# 运行
hugo

在将网站提交到gitbub之前，还需要在config.toml文件中修改baseURL:

1

baseURL = 'http://ruqing7.github.io/'

4）拓展

帮助文档： https://hexo.io/zh-cn/docs/，尤其是toml\yaml这类格式问题，分别给出了参考！

1
2

layout: false #草稿，不渲染
hugo server -D #渲染草稿文件

3. 部署到服务器

girhub 新建仓库，命名方式为

1
2

<你的github用户名>.github.io
# 例如我的：ruqing7.github.io,注意不要生成readme

创建public文件夹

1

hugo

把public与远程github仓库关联

1
2
3
4
5

git init # 初始化本地代码仓
git remote add origin git@github.com:<你的github用户名>/<你的github用户名>.github.io.git
git add . # 添加本地代码
git commit -m "first commit" # 提交本地代码
git push -u origin main # 上传代码

命令行上传完毕后，在浏览器网址栏打开链接 https://<你的github用户名>.github.io/ 就可以看到我们自己的博客了~

上传报错可以看这里： https://blog.csdn.net/qq_41999617/article/details/83691324

4. picGo

细心的你也许发现，上传的图片显示不出来。这时你也许需要搭建一个图床，这里我使用的picGo

4.1 下载

下载地址：https://github.com/Molunerfinn/PicGo/releases

下载对应版本(windows是exe)并安装到本地

4.2 搭建图床

可以参考： https://zhuanlan.zhihu.com/p/416251450

  • 如果使用typora，PicGo 路径要指定到 PicGo.exe
  • 确保PicGo的监听端口为36677
